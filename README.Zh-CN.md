<h1 align="center">Agarwood Cloud</h1>
Agarwood 是一套私域流量的解决方案，基于微信、抖音等主流平台，对粉丝做好即时接待，助力商家做好客户服务和增长

### 当前状态: Aplha

该项目还处于孵化和演进阶段，欢迎大家参与到 Agarwood 项目的建设中来！🎉🎉

- 🔥 学习最新的 `Micro Services`+`Domain Driven Design` 技术

- ⭐ 参与到开源社区中来
- 🎊 结识一群热爱学习、热爱开源的朋友
